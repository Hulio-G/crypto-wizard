# Crypto-Wizard

Crypto-Wizard is a decentralized marketing toolkit and influencer directory for Web3 narrative campaigns. It empowers both influencers and campaign creators to collaborate on crypto narratives, leveraging platforms like Lens Protocol, Nostr, Bluesky, and more.

This MVP features a visual mockup (`lens-protocal.png`) of the Lens Protocol integration as part of the "Pull Narrative" tab.

---

## 🧙 Project Vision

Crypto-Wizard will serve as:
- A platform to **push** and **pull** narratives across the crypto space
- A hub for influencers to earn through bounty-based campaigns
- A directory-driven discovery tool (Discord, Telegram, Onion, AI Tools)
- A toolkit for guerrilla marketers, content creators, and researchers
